# Synent_task4_CSV

# 📊 CSV to Interactive Dashboard Project

## 📌 Overview

This project focuses on converting a CSV dataset into an **interactive dashboard** that allows users to explore data dynamically through visualizations.

---

## 🎯 Objective

To build an interactive dashboard that enables users to upload a dataset and visualize it using dynamic charts.

---

## 🏢 Internship

This project was completed as part of an internship at **Synent Technologies**, where I gained hands-on experience in building interactive data dashboards.

---

## 📂 Dataset

* **Type:** Any dataset (e.g., Titanic, Iris, Netflix)
* Uploaded by the user in CSV format
* Used for generating dynamic visualizations

---

## 🛠️ Features Implemented

### ✅ 1. CSV File Upload

* Users can upload their own dataset
* Supports real-time data loading

### ✅ 2. Dynamic Data Display

* Displays dataset preview (head of data)
* Shows structure and summary

### ✅ 3. Interactive Visualizations

* Automatically generates charts such as:

  * Bar charts
  * Histograms
  * Scatter plots
* Updates visuals based on selected features

### ✅ 4. User Interaction

* Dropdowns for selecting columns
* Real-time updates of charts

---
